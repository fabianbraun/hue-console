Philips Hue Console

Getting started with Hue programming is simple using using jsHue and the public Philips Hue API

One thing I like about the jsHue wrapper for the Philips Hue REST API is that it is very minimal.You typically pass some JSON that you'd need to put into a PUT or POST body into a jsHue javascript function, so if the JSON spec changes, the javascript wrapper doesn't break.
